zTwitterFeed - Multiple Twitter Feeds with Ticker Animation

This example shows mutiple feeds on a single page animated by the vTicker plugin.

Script

$(document).ready(function () {
	$('#ticker1').twitterfeed('jquery',{},function(e) {
		$(e).find('div.twitterBody').vTicker({ showItems: 3});
	});

	$('#ticker2').twitterfeed('twitzazar',{},function(e) {
		$(e).find('div.twitterBody').vTicker({ showItems: 1});
	});

});
